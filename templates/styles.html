Measurements

length - specifies in unit of measurement

inherit - inherited value from parent element

Height and Width

Height and width does not include padding, borders, or margins.

height: 200px | 50% | auto | initial | inherit

width: 200px | 50% | auto | initial | inherit | max-width

Initial sets the value to it's default value

max-width: 200px

Block elements always take up the width available to them. Max-width sets the maximum size.(max-width overrides width)

Margins and Padding

Negative values are valid for margins but not padding

Margins

margin: 5px 10px 15px 20px

margin-top: 5px

margin-bottom: 5px

margin-left: 5px

margin-right: 5px

Padding

padding: 5px 10px 15px 20px

padding-top: 10px

When a container has a defined size, any padding will add to the overall size of the container. To control this you need box sizing:

div {

width: 200px;

padding: 20px;

box-sizing: border-box;

}

Borders

border-style: dotted | dashed | solid | double | groove | inset | outset | none | hidden

border-style: dotted dashed solid none

border-width: 5px | medium | thick

border-width: 5px 20px

applies 5px to the top and bottom and 20px on the sides

border-width: 5px 6px 7px 8px

5px on the top, 6px on the right, 7px on the botttom, and 8px left

border-color: red | #234 | rgb(255, 0, 100)

border-color: red green blue yellow

applies red top, green right, blue bottom and yellow left

border: 5px dashed blue

you can do width, style, colour here, but only style is required

the border shorthand can be applied to one side too, ie.border-left: 2px solid #333

border-radius: 5px

Colour and Backgrounds

Grey can be achieved by using equal values in all rgb values

The last value in rgba is the opacity. It ranges between 0 and 1.

background-image: url("cat.jpg");

background-repeat:repeat-x | repeat-y | no-repeat;

background-position: right top

background-attachment: scroll | fixed

Links

Links can be styled with any CSS property

Links have four states:

a:link {

color: blue

}

a:visited {

color: purple

}

a:hover {

color: red

}

a:active {

color: green

}


To style a link as a button:

a:link, a:visited {

background-color: white;

color: #333;

padding: 14px 25x;

text-align: center;

text-decoration: none;

display: inline-block;

}

a:hover, a:active {

background-color: #333

color: white

}

Lists

list-style-type: circle | square | upper-roman | lower-aplha;

list-style-image: url('cat.jpg');

list-style-position: outside | inside;

list-style: circle inside url("cat.jpg");

Any styles added to the list ol or ul tag will affect the whole list

  1. one
  2. two
  3. three

Any styles added to the li items will only affect the list items

  1. one
  2. two
  3. three
