after setting the position property, you can position the element using top, bottom, left and right properties

static - default positioning

relative - position relative to its normal position

fixed - position relative to the viewport (not affected by scroll)

absolute - position relative to parent container (will be removed from normal flow and can overlap elements)

sticky - relative position until it meets a given offset position, it then becomes fixed

Flexbox

Flex container properties:

display: flex

flex-direction: column | column-reverse | row | row-reverse

in which direction the container should stack the items.

flex-wrap: wrap | no-wrap | wrap-reverse

specifies whether flex items should wrap or not

flex-flow: row wrap

shorthand for settingflex-direction and flex-wrap

justify-content: center | flex-start | flex-end | space-around | space-between

align the flex items

align-items: center | flex-start | flex-end | stretch | baseline

also to align the flex items.

align-content:space-between | space-around | stretch | center | flex-start | flex-end

used to align the flex lines


Flex item properties:

order: 1 | 2 | 3 | 4 etc...

changes the order of the items

flex-grow: 1 | 2 | 3 | 4 etc...

specifies how much an item will grow relative to the other items.

flex-shrink: 1 | 2 | 3 | 4 etc...

specifies how much an item will shrink relative to the other items.

flex-basis: 200px

specifies the initial length of a flex-item

flex: 0 1 300px

shorthand for flex-grow, flex-shrink and flex-basis properties

align-self: center | flex-start | flex-end

specifies the alignment for the item inside the container. It overrides the container settings.


Responsive flex-box:

How to get items to run horizontally on laptops and stack on mobiles

.flex-container {

display: flex;

flex-direction: row;

}

@media (max-width: 800px) {

.flex-container {

flex-direction: column;

}

}

CSS Grid

A grid consists of a parent element with child elements organised in rows and columns

Grid container properties:

display: grid | inline-grid;

grid-gap: 20px | 15px 20px;

sets the gaps for the rows and columns

grid-column-gap: 30px;

grid-row-gap: 10px;

The gaps between the rows and columns are called lines. We refer to the lines when placing an item.

The far left and far top lines are column one and row one.

grid-column-start: 1;

grid-column-end: 3;

grid-row-start: 2;

grid-row-end: 5;

grid-template-columns: auto auto auto auto;

grid-template-columns: 20px auto 30px 15px;

defines the number of columns. Four x auto = equal width. Each value specifies width.

grid-template-rows: auto auto auto auto;

grid-template-rows: 20px auto 30px 15px;

defines the number of rows. Each values specifies the height.

justify-content space-evenly | space-around | space-between | center | start | end

To align the whole grid horizontally inside the container

align-items space-evenly | space-around | space-between | center | start | end

To align the whole grid vertically inside the container

Grid item properties:

grid-column: 1 / 5 | 1 / span 3 |

defines which column(s) to place an item in. Define the start line and end line.

grid-row: 1 / 5 | 1 / span 3 |

defines which row(s) to place an item in. Define the start line and end line.

grid-area: 1 / 2 / 5 / 6

shorthand for grid-row-start, grid-row-end, grid-column-start and grid-column-end
